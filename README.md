# Usage:

> [Scripts] [DICT] [MOD] [WORD|FILE]

Dictionaries:
- `-g`: German-English dictionary
- `-d`: English dictionary
- `-t`: English thesaurus
- `-e`: Look up definition of a chemical element

Modes:
- `-s`: Single object
- `-f`: File with words

## Examples:
```bash
lang -g Warum
lang -d Hello
lang -d -f Words.txt
```
